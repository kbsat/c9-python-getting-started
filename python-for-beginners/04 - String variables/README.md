# 문자열

Python은 저장하거나 문자열을 저장하거나 처리할 수 있습니다. 문자열은 작은따옴표나 큰따옴표로 감싸서 표현합니다. 문자열을 처리하는데 사용할 수 있는 여러 문자열 메소드들이 있습니다. 

- [문자열](https://docs.python.org/3/tutorial/introduction.html#strings)
- [문자열 메소드](https://docs.python.org/3/library/stdtypes.html#string-methods)

문자열 값으로 변환하기

- [str](https://docs.python.org/3/library/functions.html#func-str)

변수 이름을 정할 때 Python 코드용 PEP-8 스타일 가이드를 따릅니다.

- [PEP-8 스타일 가이드](https://www.python.org/dev/peps/pep-0008/#naming-conventions)
