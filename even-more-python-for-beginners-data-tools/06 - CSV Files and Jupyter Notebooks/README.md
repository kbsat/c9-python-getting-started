# CSV 파일과 Jupyter Notebooks

CSV 파일은 콤마로 구별된 변수 파일입니다. CSV 파일은 주로 데이터를 저장할 때 사용합니다. Jupyter Notebook에서 CSV 파일의 데이터에 액세스하려면 파일을 업로드해야합니다.\
