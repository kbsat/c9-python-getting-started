# 조건문 다루기

[if](https://docs.python.org/3/reference/compound_stmts.html#the-if-statement) 문을 이용하여 조건문을 완성할 수 있습니다.

`if` 구문

```python
if 조건:
    # 실행될 코드
else:
    # 실행될 코드
```

[비교 연산자](https://docs.python.org/3/library/stdtypes.html#comparisons)

- < ~보다 적은
- < ~보다 큰
- == ~와 같은
- \>= ~보다 크거나 같은
- <= ~보다 작거나 같은
- != ~와 같지 않은
